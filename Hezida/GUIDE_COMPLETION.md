### Pour créer un fichier en 5 min :

2. **Adapter** le contenu basé sur prompt_model.md
3. **Personnaliser** :
   - Subject line
   - Badge couleur/texte
   - H1 titre
   - Corps message (80-120 mots)
   - CTA
4. **Vérifier** :
   - CSS inline
   - Tokens `{{first_name}}` `{{company}}`
   - UTM tracking unique
   - Responsive (max-width: 100%)
